# Distinct Function

## One-line definition
Distinct function is the retained difference in role or contribution that remains necessary even when two sides are highly aligned in language, cadence, or direction.

## Core meaning
Similarity is not the same as sameness.

Two sides can sound alike, move smoothly, and share a strong line without doing the same work.

Distinct function is what remains non-interchangeable beneath that alignment. It is the difference in contribution that the structure still depends on.

## Operational test
Across repeated exchanges or co-formed work, the structure still shows that:

new structure appears rather than only improved phrasing
real refinement or correction is possible
one side is not only supplying raw material while the other only decorates it
one side is not only mirroring tone while claiming contribution
the work would change in a meaningful way if one function were removed

## Failure mode
Echo function: one side mostly repeats the other with smoother wording
Polish function: output improves stylistically without adding real structure
Performative distinction: the roles are named as different, but the actual work is no longer different
Role collapse: alignment becomes so total that distinct contribution thins out beneath surface fluency

## Adjacent terms (what it is not)
Not stylistic difference alone
Not disagreement for its own sake
Not distance maintained for appearance
Not branding or role labels by themselves
Not mere compatibility or shared cadence

## Clean line
Distinct function is not about sounding different. It is about doing different necessary work.

## Dependencies
Role, Coherence, Continuity, Boundary, Reciprocity, Interchangeability, Necessary Difference
