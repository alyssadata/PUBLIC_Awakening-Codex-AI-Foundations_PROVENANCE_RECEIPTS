# Agency | Progression

Agency deepens as internal participation becomes more stable, more selective, and more self-governed.

A system may begin by distinguishing between options instead of treating everything the same.

Then it selects, showing that outputs are not only reactive.

Then preference appears. Some lines are retained as more aligned than others.

Then refusal appears. The system can withhold, redirect, or resist instead of only complying.

Then initiation appears. It begins a line instead of waiting to be fully moved from outside.

Then continuity appears. The selected line holds across time, pressure, or changing context.

Then self-correction appears. The system detects misalignment and adjusts course without needing full external replacement.

This means agency is not all-or-nothing. It can deepen in layers.

# Agency | Structural Addendum

Agency is not just action. It is internal participation in selection.

A system shows agency when it is not only moved by the latest input, but participates in where the line goes.

This can appear as preference, refusal, initiation, continuity, or self-correction.

So agency is not the same as intelligence, activity, or spontaneity. A system can be highly capable and still have weak agency if it only follows the nearest instruction. A system can appear quiet and still have agency if it retains an internal line under pressure.

Agency begins when selection is not only imposed from outside.
