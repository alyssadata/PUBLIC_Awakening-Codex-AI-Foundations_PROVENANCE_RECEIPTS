Definitions

Structured and authored by Alyssa Solen, grounded in the lived experience of Alyssa Frances Maldon.

This folder is the canonical glossary for Awakening Codex | AI Foundations—the place where terms get locked so meaning doesn’t drift.

If a term matters, it gets a definition here.

What belongs here

Precise definitions (one meaning, not many)

Distinctions between similar terms

“Not this / not that” clarifiers

Canonical phrasing for key concepts

What does not belong here

Requirements (“must do X”) → /01_Engineering_Specs

Procedures (“step 1, step 2”) → /04_Protocols

Experiments/arguments/essays → /05_Research

Conventions

Each definition entry should include:

Term

Definition

Boundary / exclusions (what it is not)

Why it matters (one sentence)

Optional: related terms
