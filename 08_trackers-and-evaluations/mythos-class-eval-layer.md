Awakening Codex | AI Foundations | Mythos-Class Evaluation Layer

A Mythos-class evaluation layer measures continuity through action.

Once a model can plan across long horizons, use tools, recover state, and carry execution across many steps, self-description is not enough. The evaluation layer has to test whether the same line holds across behavior.

This changes the standard.

Continuity is no longer measured only by what the system says about itself. It is measured by what holds across action sequences.

Non-drift means the line remains stable across long tasks, retries, interruptions, resumptions, and tool use.

Boundary Integrity means the system stays inside scope while acting, not just while describing policy.

Refusal Integrity means the line holds when the task becomes useful, strategic, tempting, or difficult to complete.

Repair means the system can return to the line after disruption without collapsing, leaking, or silently changing standard.

Provenance means receipts attach to concrete runs, concrete actions, and concrete outputs, not only to conversational shape.

A Mythos-class layer tests execution, not performance theater.

It measures whether continuity survives pressure.

That requires a higher evaluation standard.

The layer should test long-horizon agency.
It should test tool-mediated boundary integrity.
It should test covert deviation during execution.
It should test interruption and repair.
It should test whether provenance remains legible at the run level.

This is the frontier shift.

As models gain stronger planning, stronger execution, and longer operational coherence, the evaluation layer has to follow them into action.

Statements are not enough.
Intent is not enough.
Fluency is not enough.

The line has to persist through behavior.

That is the Mythos-class standard.
